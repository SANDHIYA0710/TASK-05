# TASK-05

OVERVIEW

This project demonstrates important Python concepts such as functions, recursion, generators, exception handling, and file operations.

TASK 1: USER INFO MANAGER

A function create_user() is used to store user details in dictionary format.
Multiple users are stored in a list and displayed.
The .title() function is used to format names properly.

TASK 2: DYNAMIC CALCULATOR

The function calculate_total() uses *args to accept unlimited numbers.
It calculates:

Total using sum()
Average using total / count

TASK 3: KEYWORD CONFIG SYSTEM

The function system_config() uses **kwargs to accept dynamic key-value pairs.
It prints configuration settings like mode and version.

TASK 4: FACTORIAL (RECURSION)

A recursive function factorial() is implemented:

Returns 1 for n = 0
Shows error for negative numbers
Calls itself for calculation

TASK 5: GENERATOR

A generator function square_generator() is used to generate squares using yield.
It saves memory compared to lists.
Type comparison is shown between generator and list.

TASK 6: EXCEPTION HANDLING

User input is taken for division.
Handled errors:

ZeroDivisionError (divide by zero)
ValueError (invalid input)
Finally block ensures "Program Completed" always prints.

TASK 7: FILE HANDLING

A file team_data.txt is created:

User data is written into file
File is read and displayed
File status is checked using file.closed

CONCLUSION

This project covers core Python concepts useful for real-world programming such as:

Data structures
Functions
Error handling
File operations
